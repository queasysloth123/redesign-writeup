Selected Website

The chosen website to redesign is Craigslist

Redesigned Page

Usability Problems

  • Does not follow conventional layout of mainstream online marketplaces, which can induce difficulty in navigation
  • No visual hierarchy to highlight more important points of the page. Majority of text are the same blue and small font size
  • Does not include visual aid such as images and icons to help with navigation and keep user attention
  • Overload of information for both the subcategories and locations
  • Unable to search by categories, which can generate unwanted results
  • Very little sense of user self-control
  • How do you feel about the layout of the kiosk?
  • Repetition of links that can be consolidated into few links
  • Difficult to spot account system
  • Unable to identify the page with company name and logo

Accessibility

There were a total of two errors detected by WebAIM WAVE. The first error reports the absence of an established language of the webpage. The second error mentions that the search bar is missing a form label. These two errors are more minor technical errors that won’t necssaily negatively affect the page’s overall accessibility, as the page allows for language changes and users are still able to use the search bar normally. There was also a notable alert that describes the small font size for all the subcategories under the discussion forum category, which I agree would be unnecessarily difficult to read.
